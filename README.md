# Codeforces Tag Toggle (Chrome)

Show or hide the **Problem tags** section on Codeforces problem pages.

## What it does

- Adds a button on the right side of the **Problem tags** header
- Default state: tags are hidden
- Click **Show Tags** to reveal tags
- Click **Hide Tags** to hide tags again

## Chrome install

1. Download or clone this folder.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on **Developer mode** (top-right).
4. Click **Load unpacked**.
5. Select this folder: `cf-show-tags`.
6. Open any Codeforces problem page.
7. Use the **Show Tags / Hide Tags** button in the right sidebar.

## Works on these Codeforces URLs

- `https://codeforces.com/problemset/problem/*`
- `https://codeforces.com/contest/*/problem/*`
- `https://codeforces.com/gym/*/problem/*`
- `https://codeforces.com/group/*/contest/*/problem/*`

## If you update code later

1. Go to `chrome://extensions`
2. Click **Reload** on Codeforces Tag Toggle
3. Refresh your Codeforces page

## Notes

- This is currently prepared for Chrome/Chromium browsers (Chrome, Brave, Edge).
- Safari support can be added later via Safari Web Extension conversion.
